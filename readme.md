# css-longhand-cli

> CLI for [css-longhand](https://github.com/hemanth/css-longhand)


## Install

```
$ npm install -g css-longhand-cli
```


## Usage

```bash
$ css-longhand ./my.css ./out.css

$ css-longhand --help

  CLI for css-longhand

  Usage
    css-longhand <path/to/css> [path/to/out] --silent
```
